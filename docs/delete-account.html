账号删除

你可以在 App 内删除 Apple ID 账号关联、本机数据和开薪 iCloud 私有库记录。

最后更新：2026年5月20日

首页 隐私政策 服务条款 支持与反馈

如何删除

  1. 打开开薪 PayJoy。
  2. 进入「我的」页面。
  3. 打开「数据与同步」。
  4. 点击「删除账号与数据」，并按提示确认。

会删除什么

  • 本机保存的薪资设置、工作时间、计薪日、午休设置和加班计薪日期。
  • 本机保存的昵称、个性签名、提醒偏好和 PRO 本地状态。
  • 本机保存的 Apple ID 账号关联信息。
  • 如果你已连接 Apple ID 并使用 iCloud 同步，应用会尝试删除开薪在你 iCloud 私有数据库中的记录。

不会删除什么

删除账号不会删除你的 Apple ID，也不会影响其他 App 的 iCloud 数据。App Store 购买记录由 Apple 管理，如需退款或查看购买记录，请前往 Apple 账户相关页面处理。

无法打开 App 时

如果你无法通过 App 内入口删除数据，请通过 支持与反馈 页面提交问题，并说明你希望删除开薪 PayJoy 相关数据。
